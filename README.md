# Currency converter

React currency converter

## Tech stack

- **react**
- **typescript**
- **sass**
- **webpack**

## Commands

### Dev server
For start dev server use
```
> npm run serve
```
### Tests
For start lint tests use
```
> npm run test
```
### Build
For build use
```
> npm run build
```
Before deploy use
```
> npm run build:prod
```
